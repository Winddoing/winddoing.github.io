## ubuntu18.04引导修复

修复工具：`boot-repair`

通过ubuntu的U盘安装进入试用模式，并进行联网

```
sudo add-apt-repository ppa:yannubuntu/boot-repair
sudo apt-get update
```
```
sudo apt-get install -y boot-repair
boot-repair
```
根据提示选择yes，直到修复成功。

## ubuntu 终端乱码

> 改成全英文环境来解决乱码

在`/var/lib/locales/supported.d/en`中添加：
```
en_US.UTF-8 UTF-8
```

保存，并执行
```
sudo locale-gen
```

修改`/etc/default/locale`中的`LANG`和`LANGUAGE`，并重启系统

```
LANG="en_US.UTF-8"
LANGUAGE="en_US:en"
```
