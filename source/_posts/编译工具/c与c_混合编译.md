## 原因

### 编译器版本不兼容

添加编译选项

``` shell
-D_GLIBCXX_USE_CXX11_ABI=1
```
> cmake: `add_definitions(-D_GLIBCXX_USE_CXX11_ABI=1)`

### 未引用C++库

``` shell
-lstdc++
```
> camke: `target_link_libraries(exe_target stdc++)`
